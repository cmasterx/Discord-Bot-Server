# Discord-Bot-Server
A custom discord bot

## Setup
1. Install npm 12.x
2. cd to the repository directory
3. Run `$ npm install`
4. Create a __config.json__ file

### Config File (config.json)
```javascript
{
    "token": "Discord Bot Token Goes Here",
    "prefix": "!"
}
```

## Run
* __deploy__ `npm run start`
* __dev__ `npm run dev`
